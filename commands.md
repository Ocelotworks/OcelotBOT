# OcelotBOT Commands

### Fun
 - **!8ball**
 	- Description: Ask the magic 8ball a question.
 	- Usage: `!8ball <question>`
 	- Example: 
 		 >!8ball Do My Parents Love Me?

- **!meme**
	- Description: Add a meme to the bot to refer to later.
	- Usage: `!meme <name>/list/<add> <URL>
	- Example:
		> !meme ocelotbot  
		> !meme add meirl https://i.redd.it/bvx4ifhugl5z.jpg  
		> !meme list

- **!pornsuggest**
	- Description: Suggest a style of porn for your own pleasure.
	- Usage: `!pornsuggest [country]`
	- Example:
		> !pornsuggest  
		> !pornsuggest gb

- **!trivia**
	- Description: Get a random trivia question.
	- Usage: `!trivia`
	- Example: 
		> !trivia

- **!insult**
	- Description: Savagely roast one of your bros.
	- Usage: `!insult <user>`
	- Example:
		> !insult @everyone
	
- **!numberfact**
	- Description: Get a thrilling fact about a number
	- Usage: `!numberfact <number>`
	- Example:
		> !numberfact 420

- **!spongebob**
	- Description: Creates a SPonGeBOb meme, with no arguments it copies the text above it
	- Usage: `!spongebob [text]`
	- Example:
		> !spongebob Ocelotbot is Great  
		> !spongebob

- **!crush**
	- Description: Show that you have a crush on another user, or input an image URL.
	- Usage: `!crush <@user/URL>`
	- Example:
		> !crush @OcelotBOT  
		> !crush https://upload.wikimedia.org/wikipedia/en/thumb/4/47/Spongebob-squarepants.svg/666px-Spongebob-squarepants.svg.png


- **!spell**
	- Description: Spell a sentence using reactions. Put ^ at the start to spell it on the last message.
	- Usage: `!spell [^] <sentence>`
	- Example:
		> !spell Hello World  
		> !spell ^ Stupid Example

- **!expand**
	- Description: E x p a n d  a sentence
	- Usage: `!expand <sentence>`
	- Example:
		> !expand AESTHETIC


### Utility 

- **!define**
	- Description: Define a word
	- Usage: `!define <word>`
	- Example:
		> !define example

- **!defineud**
	- Description: Define a word with Urban Dictionary
	- Usage: `!define <word>`
	- Example:
		> !defineud Dutch Oven

- **!image**
	- Description: Gets a random image post from a random subreddit
	- Usage: `!image <subreddit>`
	- Example:
		> !image aww  
		> !image random

- **!ping**
	- Description: Ping an IP address or website
	- Usage: `!ping <IP>`
	- Example:
		> !ping google.com  
		> !ping 216.58.214.14

- **!ipinfo**
	- Description: Get information about an IP address like domain and location
	- Usage: `!ipinfo <IP>`
	- Example:
		> !ipinfo 216.58.214.14

- **!whois**
	- Description: WHOIS lookup a domain
	- Usage: `!whois <domain>`
	- Example:
		> !whois unacceptableuse.com

- **!holiday**
	- Description: Check for an national holday in a set country
	- Usage: `!holiday <country-code>`
	- Example:
		> !holiday us  
		> !holiday gb

- **!flip**
	- Description: Flips a coin...
	- Usage: `!flip`
	- Example:
		> !flip

- **!remind**
	- Description: Set a reminder for some point in time
	- Usage: `!remind <in> "message"`
	- Example:
		> !remind in 10 days "sort out life"

- **!commands**
	- Description: Gives a link to what you're looking at right now
	- Usage: `!commands`
	- Example:
		> !commands

- **!translate**
	- Description: Translates something into a chosen language.
	- Usage: `!translate <language-code> <sentence>`
	- Example:
		> !translate en merde  
		> !translate fr cheese omlette

- **!weather**
	-  Description: Get the weather for a place
	-  Usage: `!weather <place>`
	-  Example:
		> !weather Egypt  
		> !weather Paris, France

- **!rlive**
	- Description: Subscribe to a reddit live thread
	- Usage: `!rlive subscribe/unsubscribe <url>`
	- Example:
		> !rlive subscribe https://www.reddit.com/live/z4vlt7665iz8  
		> !rlive unsubscribe

- **!youtube**
	- Description: Downloads the audio from a youtube video
	- Usage: `!youtube <url>`
	- Example:
		> !youtube https://www.youtube.com/watch?v=dQw4w9WgXcQ

- **!time**
	- Description: Shows you the current server time, or a timezone of your choosing
	- Usage: `!time [timezone]`
	- Example:
		> !time  
		> !time Europe/Berlin


### Meta

- **!invite**
	- Description: Generate a link to invite this bot to other servers
	- Usage: `!invite`
	- Example:
		> !invite 

- **!info**
	- Description: Get technical info about things
	- Usage: `!info bot/spell/server/channel`
	- Example:
		> !info server  
		> !info bot

- **!status**
	- Description: Shows the status of OcelotBOT
	- Usage: `!status`
	- Example:
		> !status

- **!peterstate**
	- Description: Shows the status of Peter
	- Usage: `!peterstate`
	- Example:
		> !peterstate

- **!nowplaying**
	- Description: Shows what Peter is listening to
	- Usage: `!nowplaying`
	- Example:
		> !nowplaying


NOTE: The bot also responds to messages mentioning it, if a nickname is set for it
